## IF THINGS GO WRONG

### Issue: Modal won't close
**Action:** Click X button  
**Fallback:** Press Escape key  
**Last Resort:** Refresh page (F5)

### Issue: Create button is disabled
**Check:** Are all required fields filled?  
**Solution:** Fill in all fields, button will enable  
**Note:** Invoice # and Party Name are required

### Issue: Data didn't save
**Check:** Did you click "Create" button?  
**Solution:** Try creating again, watch for modal close  
**Note:** If still doesn't work, refresh and check console

### Issue: Financial data looks wrong
**Acknowledge:** "These are demo numbers for illustration"  
**Explain:** "The formulas are correct - calculations will be accurate with real data"  
**Move On:** "Let me show you another feature..."

### Issue: Page is slow
**Cause:** Might need to wait for load  
**Action:** Give it a moment, then continue  
**Note:** All data is pre-loaded, should be fast
